MightyBlocks focuses on making it easier for website owners to add elements. All elements contains super flexible customization options.

* Content Box
* Button
* Icon
* Video (Youtube/Vimeo)

== Installation ==

1. Install Gutenberg plugin.
2. Activate the plugin. This will enable the MightyBlocks for Gutenberg.
3. Done
